## CounterCV
Desktop card counting assistant for virtual live-dealer blackjack. Built with Electron, React, and Typescript.

Currently using the Zen Counting system as default and initial implementation due to high betting correlation.

## Features
*Work in progress

- Counting engine - True count with configurable deck amount
- Action recomendation - Optimal action given true count
- Bet Spread - Customizable bet ramp along with recomended bet amount using Kelly Criterion
- Illustrious 18 alert - alerts when to deviate from basic strategy
- Screen Capture - Count without having to enter drawn cards
